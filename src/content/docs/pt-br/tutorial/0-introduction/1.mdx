## O que preciso saber para começar?

Se você tem alguma familiaridade básica com **HTML**, **Markdown**, **CSS** e um pouco de **JavaScript**, então você está totalmente preparado! Você será capaz de completar todo o tutorial apenas seguindo as instruções. Astro é para todos! 🧑‍🚀 👩‍🚀 👨‍🚀 

Você também precisará de uma conta no [GitHub](https://github.com) (ou similar) para publicar seu projeto na web.

<details>
<summary>Como uso as listas de verificação no final de cada página?</summary>

Você as marca!

No final de cada página, você encontrará uma lista de tarefas clicável das atividades que agora deve ser capaz de fazer. Marque esses itens para ver seu progresso no Rastreador de Tutorial.

Usar o rastreador é opcional, mas pode ajudá-lo a lembrar seu lugar se você completar o tutorial em várias visitas. Você também pode deixar algumas listas em branco como lembrete das unidades que deseja revisitar mais tarde.

(Esses dados são salvos apenas no armazenamento local do seu navegador e não estão disponíveis em outros lugares. Nenhum dado é enviado ou armazenado pelo Astro.)
</details>

<details>
<summary>A Unidade 1 é sobre coisas que já sei fazer. Posso pular?</summary>

Você pode usar as lições dentro da [Unidade 1](/en/tutorial/1-setup/) para garantir que possui as ferramentas de desenvolvimento e contas online necessárias para completar o tutorial. Ele o guiará na criação de um novo projeto Astro, armazenando-o no GitHub e implantando-o no Netlify.

Se você [criar um novo projeto Astro vazio](/en/install/auto/) e estiver confortável com sua configuração, pode pular com segurança para a [Unidade 2](/en/tutorial/2-pages/) onde começará a criar novas páginas em seu projeto.
</details>
