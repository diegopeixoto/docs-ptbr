## Obtenha as ferramentas de desenvolvimento necessárias

### Terminal

Você usará uma **linha de comando (terminal)** para criar o seu projeto com Astro e para executar comandos essenciais para construir, desenvolver e testar o seu site.

Você pode acessar a linha de comando através de um programa de terminal local para o seu sistema operacional. Aplicativos comuns incluem **Terminal** (MacOS/Linux), **Prompt de Comando** (Windows) e **Termux** (Android). Provavelmente, um desses já está no seu computador.

### Node.js

Para que o Astro funcione no seu sistema, você também precisa ter o [**Node.js**](https://nodejs.org/pt-br/) instalado, versão `v16.12.0` ou posterior.

Para verificar se você já tem uma versão compatível instalada, execute o seguinte comando no seu terminal:

node -v

// Exemplo de saída
v16.14.0


Se o comando retornar um número de versão maior que `v16.12.0`, está tudo certo!

Se o comando retornar uma mensagem de erro como `Comando 'node' não encontrado` ou um número de versão inferior a `v16.12.0`, você precisará [instalar uma versão compatível do Node.js](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm).

### Editor de Código

Além disso, você precisará baixar e instalar um **editor de código** para escrever o seu código.

:::tip[usaremos...]
Este tutorial usará o **VS Code**, mas você pode usar qualquer editor para o seu sistema operacional.
:::
