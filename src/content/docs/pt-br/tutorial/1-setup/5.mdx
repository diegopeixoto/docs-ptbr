Aqui, você irá conectar o seu repositório do GitHub ao Netlify. O Netlify usará esse projeto para construir e implantar o seu site ao vivo na web toda vez que você fizer um commit em uma alteração no seu código.

:::tip[Nós usaremos...]
Este tutorial usará o **Netlify**, mas você é bem-vindo(a) a usar o serviço de hospedagem de sua preferência para implantar o seu site na internet.
:::

## Crie um novo site no Netlify

1. Crie uma conta gratuita no [Netlify](https://netlify.com) se você ainda não tiver uma.

    Anote o seu nome de usuário. Você visualizará o seu painel e qualquer site que criar em `https://app.netlify.com/teams/username`

2. Clique em <kbd>Adicionar novo site</kbd> > <kbd>Importar um projeto existente</kbd>.

    Você será solicitado(a) a conectar-se a um provedor Git. Escolha o GitHub e siga as etapas na tela para autenticar a sua conta do GitHub. Em seguida, escolha o repositório do seu projeto Astro no GitHub a partir da lista fornecida.

3. Na etapa final, o Netlify irá mostrar as configurações do site do seu aplicativo. As configurações padrão devem estar corretas para o seu projeto Astro, então você pode rolar a página para baixo e clicar em <kbd>Implantar site</kbd>.

Parabéns, você tem um site Astro!

## Mude o nome do seu projeto

Na página de visão geral do seu site no Netlify, você verá o nome do seu projeto gerado aleatoriamente e a URL do seu site com o formato `https://project-name-123456.netlify.app`. Você pode mudar o nome do seu projeto para algo mais memorável, e isso atualizará automaticamente a sua URL.

## Visite o seu novo site

Clique na URL nas configurações do seu site ou digite-a em uma janela do navegador para ver o seu novo site.
