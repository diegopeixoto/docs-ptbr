Agora que você sabe o que vai construir, vamos configurar todas as ferramentas de que você vai precisar!

Esta unidade mostra como configurar o seu ambiente de desenvolvimento e implantar no Netlify. Vá direto para [Unidade 2](/pt-br/tutorial/2-pages/) se você já estiver confortável com o seu ambiente e fluxo de trabalho.

:::tip[usando o StackBlitz]

Quer completar este tutorial em um editor de código online?
<details>
<summary>Siga estas instruções e vá diretamente para a Unidade 2!</summary>

**Configurar o StackBlitz**
1. Visite [astro.new](https://astro.new) e clique no botão para abrir o modelo "Empty Project" no StackBlitz.

2. Clique em "Sign in" no canto superior direito para fazer login usando suas credenciais do GitHub.

3. No canto superior esquerdo da janela do editor do StackBlitz, clique em "fork" do modelo (salve-o no seu próprio painel de conta).

4. Aguarde o carregamento do projeto e você verá uma prévia ao vivo do modelo "Empty Project".

**Fazer uma mudança**

No painel de arquivos, você deve ver `src/pages/index.astro`. Clique para abri-lo e siga [Escreva a sua primeira linha de código Astro](/pt-br/tutorial/1-setup/3/) para fazer uma mudança neste arquivo.

**Criar um repositório no GitHub**

1. Pressione o botão <kbd>Connect Repository</kbd> na parte superior da lista de arquivos, insira um novo nome para o seu repositório e clique em <kbd>Create repo & push</kbd>. 

2. Quando você tiver alterações a serem enviadas de volta para o GitHub, um botão "Commit" aparecerá no canto superior esquerdo do seu espaço de trabalho. Clicar nele permitirá que você insira uma mensagem de commit e atualize o seu repositório.

**Implantar o seu Site**

Se você deseja implantar no Netlify, vá para [Implante o seu site na web](/pt-br/tutorial/1-setup/5/).
Caso contrário, vá para [Unidade 2](/pt-br/tutorial/2-pages/) para começar a construir com Astro!

</details>
:::

## Para onde estamos indo?

Nesta unidade, você **criará um novo projeto** que será **armazenado online no GitHub** e **conectado ao Netlify**.

Conforme você escreve o código, você fará commits periódicos das suas alterações no GitHub. O Netlify usará os arquivos no seu repositório do GitHub para construir o seu site e, em seguida, publicá-lo na internet em um endereço único onde qualquer pessoa poderá visualizá-lo.

Cada vez que você faz um commit no GitHub, uma notificação é enviada para o Netlify. Em seguida, o Netlify reconstruirá e republicará automaticamente o seu site ao vivo para refletir essas alterações.
